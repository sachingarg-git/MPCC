1. Application Overview

The MPCC (Medical Pollution Control Committee) Application is a comprehensive healthcare facility management system designed to streamline the registration, monitoring, and management of Healthcare Facilities (HCFs). The system provides complete lifecycle management from registration to renewal and de-registration.

🏥

HCF Management

Complete 360° view of all healthcare facilities with document management, contact tracking, and status monitoring.

📋

Registration Workflow

Streamlined registration process with approval workflows, document verification, and certificate generation.

💳

Payment Tracking

Complete payment management with invoice generation, payment recording, and outstanding tracking.

📊

Reports & Analytics

Comprehensive dashboard with real-time statistics, status breakdowns, and performance metrics.

Key Features

  • HCF Master Module: Complete healthcare facility database with 360° view
  • Document Repository: Upload and manage all HCF-related documents
  • Payment Management: Track payments, invoices, and outstanding amounts
  • Renewal Management: Automated renewal tracking and processing
  • De-registration: Manage facility closures and reactivations
  • Customer Portal: Self-service portal for HCF customers
  • Role-based Access: Secure access control with customizable roles

2. Getting Started

2.1 Login Process

Access the Application

Open your web browser and navigate to the application URL. You will see the landing page with login options.

Enter Credentials

Click on "Admin Login" or "Staff Login" and enter your username and password provided by the system administrator.

Access Dashboard

Upon successful authentication, you will be redirected to the main dashboard based on your role permissions.

💡 Tip If you forget your password, contact your system administrator for a password reset. Do not share your login credentials with others.

2.2 Navigation Guide

The application uses a sidebar navigation system with the following main sections:

Menu Item Icon Description
Dashboard 📊 Overview of all system metrics and recent activities
CRM / HCF 🏥 Healthcare facility management and CRM functions
Customer 👥 Customer registration and management
Master Data 📋 System configuration and master data management
Settings ⚙️ User management, roles, and system settings

3. Dashboard

The Dashboard provides a comprehensive overview of the system's key metrics and recent activities.

Dashboard Sections

👥 Customer Overview

  • Total Registrations: Total number of HCFs registered in the system
  • Approved / Active: Number of active facilities
  • Pending Review: Registrations awaiting approval
  • Total Revenue: Sum of all payments received

📋 Master Data Overview

  • Zones: Active zones out of total zones configured
  • Routes: Active collection routes
  • Service Plans: Available service plans
  • Kits: Waste collection kits in inventory

🏥 CRM / HCF Overview

  • Categories: HCF facility categories
  • Raw Materials: Materials in master database
  • HCF Registered: Total facilities across all zones
  • This Month: New registrations in current month

4. CRM / HCF Master Module

The CRM/HCF module is the core of the application, providing complete management of healthcare facilities.

4.1 HCF Master - 360° View

The HCF Master section provides a comprehensive view of all healthcare facilities with the following features:

Stats Cards

Quick overview showing:

  • All HCFs: Total count of all facilities
  • Active: Currently active facilities
  • Due for Renewal: Facilities approaching renewal date
  • Defaulters: Facilities with outstanding payments
  • Deregistered: Closed or inactive facilities

HCF Table

The main data table displays:

Column Description
Institution NameName of the healthcare facility
CategoryType of facility (Hospital, Clinic, etc.)
Zone / RouteAssigned zone and collection route
Contact PersonPrimary contact at the facility
MobileContact phone number
StatusCurrent status (Active, Suspended, etc.)
Reg. DateOriginal registration date
Renewal DateNext renewal due date
OutstandingPending payment amount
DocsQuick access to documents
ActionsView 360°, Edit, Delete options

360° View Modal

Clicking the "360°" button opens a detailed view with tabs:

  • Overview: Basic facility information and quick stats
  • Payments: Payment history and invoice management
  • Pickups: Waste collection schedule and history
  • Documents: Document repository with upload/view
  • Certificates: Generated certificates
  • Complaints: Support tickets and issues
  • Contacts: Contact persons at the facility
  • Timeline: Complete activity history

Document Repository

The Documents tab allows management of:

  • Aadhaar Card
  • PAN Card
  • GST Certificate
  • BMW Authorization
  • PCB Authorization
  • Cancelled Cheque
  • Facility Photo
  • Letterhead
  • MoU Copy
  • Agreement Copy
  • NOC (CMO/RO Consent)
✅ Document Status Documents show status badges: Valid (green), Expiring Soon (yellow), Expired (red), or Not Uploaded.

4.2 De-registration Management

This section manages facility closures and status changes:

Status Buckets

  • Pending Requests: New de-registration requests awaiting review
  • Under Review: Requests being processed
  • Approved: Approved de-registrations
  • Rejected: Rejected requests
  • Completed: Fully processed de-registrations

Actions Available

  • Review: Open detailed review checklist
  • Approve: Approve the de-registration request
  • Reject: Reject with reason
  • Reactivate: Reactivate a de-registered facility

History Tracking

All de-registration actions are tracked with:

  • Action taken (Approve, Reject, Reactivate)
  • User who performed the action
  • Timestamp
  • Remarks/Notes

4.3 Renewal Management

Manage contract renewals for healthcare facilities:

Renewal Status

  • Due for Renewal: Facilities with contracts expiring soon
  • Renewal Pending: Renewal initiated but not completed
  • Renewed: Successfully renewed contracts
  • Expired: Contracts that have expired

Renewal Process

Identify Due Renewals

Filter HCFs by renewal date or use the "Due for Renewal" bucket.

Initiate Renewal

Click the renewal button to start the process.

Update Contract Details

Enter new contract dates, fees, and terms.

Generate Documents

Generate new MoU and certificates as needed.

Configure system-wide master data used across the application.

Available Masters

🗺️

Zone Master

🛣️

Route Master

Configure collection routes within zones.

🏷️

Category Master

Service Plans

7. User & Role Management

User Management

Administrators can manage system users:

  • Create new user accounts
  • Assign roles and permissions
  • Enable/disable user accounts
  • Reset passwords
  • View user activity logs

Role Management

Configure role-based access control:

Role Typical Permissions
Super Admin Full access to all modules and settings
Admin Manage users, approve registrations, full CRM access
Manager View reports, manage assigned zones, approve requests
Operator Data entry, basic CRM functions, view only reports
Viewer Read-only access to reports and data

8. Customer Portal

The Customer Portal provides self-service access for registered healthcare facilities.

Portal Features

  • Profile Management: View and update facility information
  • Document Upload: Submit required documents online
  • Payment History: View invoices and payment records
  • Certificate Download: Download registration certificates
  • Support Tickets: Raise and track support requests
  • Renewal Requests: Initiate contract renewals

Accessing the Portal

Navigate to Portal

Access the portal URL: /portal

Login with OTP

Enter your registered mobile number and verify with OTP.

Access Dashboard

View your facility dashboard with pending actions and notifications.

9. Troubleshooting

Common Issues

⚠️ Unable to Login
  • Verify your username and password are correct
  • Check if your account is active
  • Clear browser cache and cookies
  • Contact administrator if issue persists
⚠️ Page Not Loading
  • Check your internet connection
  • Refresh the page (Ctrl + F5)
  • Try a different browser
  • Clear browser cache
⚠️ Document Upload Failed
  • Check file size (max 5MB recommended)
  • Ensure file format is supported (PDF, JPG, PNG)
  • Try compressing the file
  • Check internet connection stability
